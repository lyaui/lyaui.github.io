什麼是 This

每當函數被執行，執行環境被創造，同時 JavaScript 引擎會給我們一個不曾宣告的東西： this 變數。
this 會指向不同的物件，並依據函數 如何被呼叫而 改變。

  • this 是 JavaScript 的一個關鍵字。
  • this 是 function 執行時，自動生成的一個內部物件。
  • 隨著 function 執行場合的不同，this 所指向的值，也會有所不同。
  • 在大多數的情況下， this 代表的就是呼叫 function 的物件 (Owner Object of the function)。
    出自重新認識 JavaScript: Day 20 What’s “THIS” in JavaScript (鐵人精華版)

This 的各種指向

This 在 window

在 window 中，this 會指向全域也就是 window。

This 在 function

在 function 中，無論陳述或表達式，this 會指向全域也就是 window

This 在 method

this 會指向其所屬物件

This 在 method 中的 function

此時 this 不是指向所屬物件，而是指向 window。

為了修正上述問題，可以透過定義一個變數 that (self, vm…) ，讓 const that = this 來傳參考，此時 that 會指向和 this 一樣的物件，也就是保存

關於物件 this 詳細的綁定方法會在之後詳細討論。
