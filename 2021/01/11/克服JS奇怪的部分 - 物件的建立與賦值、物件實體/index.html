物件的建立與賦值

物件是一群「名稱/值」的組合，而這些值也可以是另一個「名稱/值」的組合。而物件的值可以是：

  • 純值（屬性）
  • 其他的物件（屬性）
  • 函式（方法）

物件在記憶體中有個位置，能夠 參考 到其他和它相關東西的記憶體位置，所以它知道那些不同的屬性（純值、其他物件）和方法（函式）在哪。

物件建立

  • 使用 new Object() 建立新的物件（不推薦）
  • 物件實體 (Object Liter)：直接使用大括號{}建立物件（較推薦）

物件新增值

  • 使用 物件[名稱] 新增值
  • 使用 物件.名稱 新增值（較推薦）

偽裝命名空間 (Namespace)

A container for variables and functions

Typically to keep variables and functions with the same name separate.
一個包裝物、一個容器，通常用來維持變數和函數的名稱分開。
