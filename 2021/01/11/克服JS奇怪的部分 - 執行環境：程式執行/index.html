克服JS奇怪的部分 - 執行環境：程式執行

Disqus:

語法解析器(Syntax parts)

A program that reads your code and determines what it does and if its grammar is valid.
Your code isn’t magic. Someone else wrote a program to translate it for the computer.

將你的 code 轉換成電腦能夠讀取理解的指令。

詞彙環境(Lexical environments)

Where something sits physically in the code you write.
A lexical environment exists in programming languages in which where you write something is important.

程式碼被寫在哪裡？它的周圍環境是什麼？

執行環境/ 執行脈絡/執行上下文(Execution context)

A wrapper to help manage the code that is running.
There are lots of lexical environments. Which one is currently running us managed via execution contexts. It can contain things beyond what you’ve written in your code. （因為程式碼正被轉換成電腦的語言）

一個包裹，幫助管理正在執行的程式。
