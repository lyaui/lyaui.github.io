Disqus:

表達式 (Expression)

A unit of code that result un a value.

It doesn’t have to save to a variable.
表示式是程式碼的單位，會形成/回傳一個值，而這個值不一定要儲存在某個變數。

陳述式 v.s. 表達式

陳述式 表示式

Hoisting V X

陳述式 與 表達式 的提升 (Hoisting)

  • 陳述式
    已於記憶體預留空間宣告為 fn，也就是說函數已經全部在記憶體中了，所以當程式被逐行執行時可以找到它們，因此函數裡面的程式碼可以被執行。

  • 表示式
    匿名函式被以變數的方式宣告，而變數在創造階段時會在記憶體中被設定成預設值 undefined，直到執行的時候才會被賦予成函式。

    因此若在賦值之前先執行的話，等同執行 undefined，但因為 undefined 並不是函式，故會得到 Undefined is not a function 的警告。

＊一級函數概念的運用：函數就是物件，傳入別的地方可以很快被創造和使用， 把函數給另一個函數，就像使用變數一樣，引入一個全新的型態到程式語言中，稱為函數程式語言 （Functional Programming）
