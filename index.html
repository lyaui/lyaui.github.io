Disqus:

參數 (Arguments)

The parameters you pass to a function

JavasScript gives you a keyword of the same name which contains them all
傳入函數的變數的另一個稱呼而已，可以稱呼它為 parameters，也可以稱它為 arguments。

arguments 包含了所有傳入函數的參數。

參數也會提升 (Hoisting)，當函數被執行時，它會預先設定好記憶體空間給參數，並設定它們為 undefined

傳入參數會由左至右處理

當沒有參數傳入時使用預設參數

ES6 中的預設參數

函數中的關鍵字 Arguments

argument 物件是一個對應傳入函式之引數的類陣列（Array-like）物件。

＊將 array -like 物件轉換為 array 的方法：

  • Array.prototype.slice.call(arguments)
  • Array.from(arguments)
  • [...arguments]

克服JS奇怪的部分 - This

Disqus:

什麼是 This

每當函數被執行，執行環境被創造，同時 JavaScript 引擎會給我們一個不曾宣告的東西： this 變數。
this 會指向不同的物件，並依據函數 如何被呼叫而 改變。

  • this 是 JavaScript 的一個關鍵字。
  • this 是 function 執行時，自動生成的一個內部物件。
  • 隨著 function 執行場合的不同，this 所指向的值，也會有所不同。
  • 在大多數的情況下， this 代表的就是呼叫 function 的物件 (Owner Object of the function)。

克服JS奇怪的部分 - 傳值和傳參考

Disqus:

純值傳值 (by value)

拷貝一個值到另一個不同的記憶體地點

設定一個新的變數 b，令 b = a，若為純值，b 會指向一個拷貝的 新的記憶體，填上相同的值，這個方式叫傳值。
若 a 或 b 的值受到改變，因為是不同的記憶體所以不會影響到對方。
