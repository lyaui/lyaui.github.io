克服JS奇怪的部分 - 傳值和傳參考

Disqus:

純值傳值 (by value)

拷貝一個值到另一個不同的記憶體地點

設定一個新的變數 b，令 b = a，若為純值，b 會指向一個拷貝的 新的記憶體，填上相同的值，這個方式叫傳值。
若 a 或 b 的值受到改變，因為是不同的記憶體所以不會影響到對方。

動態型別 (Dynamic Typing)

You don’t tell the engine what type of data a variable holds, it figures it out while your code is runnung

一變數可以在不同時候擁有不同型別，不用告訴 JavaScript 變數是何種型別的資料，它會在程式 執行時 自動知道。
